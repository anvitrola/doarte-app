# Requeriments

- [Git](https://git-scm.com/) installed
- [Node](https://node.js.org/) installed
- Npm or [Yarn](https://yarnpkg.com/) installed

# Initializing the app

- Run `npm install` or `yarn install`.
- Run `yarn dev` and access `http://localhost:3000`
